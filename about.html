School Projects

  1. Test Case Generator
    • It takes as its argument a file that has a Backus-Naur Form grammar
    • The form includes specifications on the kind of test cases to be generated like:
      • The number of randomly generated test-cases
      • The range of values to consider if desired test cases are integers
      • A string of possible chars if test case should be a string
    • The outcome of the project was a program that creates concise test-cases for functions
    • Languages used: Java, python
  2. Speech Tagging
    • The program used Mackov chain model to analyze the probability of a two or three speech tags following each other
    • We then used the generated probabilities to try and assign speech tags to words in a documents
    • Finaly we measured the accuracy of the model when implemented using two transitions or three transitions
    • Languages used: Python
